It is necessary to create a Telegram bot that will monitor the current rate of cryptocurrencies. The user will have the option to add a tracking currency with a price that should trigger the notification.

- Cryptocurrency tracking. For example, a /track command with BTC and a target price of 55,000 means that the user wants to track the price of bitcoin and be notified when the price reaches 55,000 USD.

— List of monitored cryptocurrencies of the user and their target prices

— Deleting tracked cryptocurrency

- Update target price. For example, /update with BTC coin and target price of 60000 means that the user wants to update the target price for bitcoin to 60.000 USD

— Checking the current price of the selected cryptocurrency

- Disable notifications for a specific cryptocurrency

— List of available commands

Conditions:

1. The service must run in Docker

2. Use as a platform and programming language: Node.js + Typescript

3. Library for developing bots: Telegraf.js (https://telegraf.js.org/)

4. Database: PostgreSQL

5. ORM for database interaction: MikroORM

6. You need to keep a log of all actions and requests

7. The rate of cryptocurrencies must be updated in real time using any third-party API (for example, CEX, CoinBase, Binance, etc.)

8. The source code, the Dockerfile for building the image and the README for using the bot must be placed on a public service (for example, github)
